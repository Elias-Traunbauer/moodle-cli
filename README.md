# Moodle CLI

Simple utility to fetch the moodle api.

![MoodleCLIInProgress](moodle-cli.gif)

## Getting started

Define two Environment variables:

 - MOODLE_USER
 - MOODLE_PASSWORD
 
 Start the CLI without any paramters.
